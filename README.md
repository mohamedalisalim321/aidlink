# 🚨 AidLink — Offline Emergency Guide App

AidLink is a fast, simple, and reliable **offline emergency assistance app** that provides step-by-step guidance for life-threatening situations.  
Designed for panic moments, AidLink works without internet and helps users take the right action quickly.

---

## 📱 Features

### 🆘 Emergency Guidance
- Step-by-step instructions for critical situations
- Clear illustrations for every step
- Simple language for quick understanding

### 🔊 Voice Guidance
- Read-aloud instructions
- Helpful when hands or eyes are busy
- Can be enabled or disabled in settings

### 📶 Fully Offline
- No internet required
- Works anywhere, anytime
- Perfect for remote or low-signal areas

### 🚑 Quick Emergency Call
- One-tap emergency call button
- Always accessible from main screens

### 🎨 Modern UI
- Clean medical-style design
- Smooth animations
- Calm color palette for stressful moments
- Dark Mode support

### ⚙️ Customization
- Light / Dark theme
- Language selection (more coming soon)

- 😮‍💨 Breathing Problems
- 🔥 Burns

- 🧠 Stroke
- ☠️ Poisoning
- ⚡ Electric Shock
- 🍽 Choking
- ☀️ Heat Stroke
- 🐍 Snake Bite
